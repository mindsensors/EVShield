§ getButtonState()

bool EVShield::getButtonState ( uint8_t  btn)

Get the button state of the specific button on EVShield.
When using the Wi-Fi Arduino Interface for PiStorms, there is only a GO button. The PiStorms does not have a left or right button like the EVShield. In this case, the F1 software button will be BTN_LEFT, and F2 will be BTN_RIGHT. If a program asks you to press the left button, instead tap the stylus in the F1 area on screen.

Parameters
btnButton to get state for (BTN_GO, BTN_LEFT, BTN_RIGHT)
Returns
true or false for specified button on the EVShield
